6.What is the difference between method overloading and method overriding in Java?
a) Method overloading allows a subclass to provide a specific implementation of a method, while method overriding does not.
b) Method overloading occurs when two methods in the same class have the same name but different parameters, while method overriding occurs when a subclass provides a specific implementation for a method defined in its superclass.
c) Method overloading and method overriding are the same concepts in Java.
d) Method overloading allows a class to inherit methods from its superclass.

Answer:b) Method overloading occurs when two methods in the same class have the same name but different parameters, while method overriding occurs when a subclass provides a specific implementation for a method defined in its superclass.
